VCube assíncrono com falsas suspeitas

Para simular a falsa suspeita, quando um processo correto i testa um processo correto j, é calculado com base em uma certa probabilidade se vai ocorrer uma falsa suspeita, ou seja, o processo i pensa que o processo j está falho.

Função que calcula uma falsa suspeita (implementada com a função randomic de rand.c disponibilizada pelo professor): 

int falsa_suspeita(int prob)
{
  return randomic(1, 100) <= prob;
}

Basicamente é gerado um número aleatório equiprovável de 1 a 100, e verificado se o valor gerado é menor que a probabilidade escolhida. (Ex.: prob = 30, significa que a função retorna 1 caso seja escolhido um valor de 1 a 30).

Além disso, quando o processo correto j descobre que foi vítima de uma falsa suspeita, ele encerra sua execução imediatamente (falha).

Código-fonte (arquivos alterados em relação ao Trabalho 1):

  • smpl.h (correção: export do símbolo random para randomic, pois em rand.c é randomic)
